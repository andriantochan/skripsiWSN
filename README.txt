Cara penggunaan program:
1. Setup sensor node sebagai base station di file BaseStation.java bagian attribut "curr"
2. Setup sensor node sebagai node di file SensorManager.java bagian attribut "sensorId", "SENSOR_NODE_ADDRESS", "PREVIOUS_NODE_ADDRESS", "NEXT_NODE_ADDRESS", "BASE_STATION_ADDRESS"
2. Jalankan Sensor_demo.jar
3. Check Online dengan input '1'
4. Jika semua sensor sudah "ONLINE", masukan input '2'. akan muncul window untuk menampilkan grafik hasil amplitudo dan frekuensi
5. Untuk memberhentikan sensing, masukan input '3'
6. Untuk memberhentikan program, masukan input '4'
